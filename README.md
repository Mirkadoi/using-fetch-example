Проект создан с использованием [Create React App](https://github.com/facebook/create-react-app).

## Описание

Репазиторий сайта по проверки тестов по принципу ключ-ответ. На сайт грузится тест(ключи) и решение, после чего выдается результат проверки:  результат(в процентах) и ошибки.

## Устройства

Сайт работает на Desktop устройствах (safari, google chrome, edge), последней версии. Мобильные устройства - опционально, работа не гарантируется.

## Доступные скрипты проекта

### `npm start`

Запуск проекта в режиме разработчика.<br />
Откройте [http://localhost:3000](http://localhost:3000) что бы увидеть приложение в браузере.

### `npm run build`

Соберает проект в продакшен моде. Результат в папке `build` .<br />
